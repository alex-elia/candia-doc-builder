# Satellite AI Presentation Example

This folder contains a complete example of using Candia Doc Builder to generate a professional pitch presentation for an embedded AI satellite system.

## What's Included

- **`PITCH_README.md`** - Documentation for the LaTeX pitch document
- **`generate_thales_presentation.py`** - Main presentation generator
- **`generate_thales_presentation_enhanced.py`** - Enhanced version with additional features
- **`generate_with_custom_template.py`** - Custom template example
- **`generate_with_premium_template.py`** - Premium template example
- **`generate_with_template_correct.py`** - Template-based generation with proper placeholders

## Usage

```bash
# From the repository root
cd examples/satellite
python generate_thales_presentation.py
```

## Purpose

This example demonstrates:
- Generating professional PowerPoint presentations from Python
- Using LaTeX for high-quality PDF documents
- Template-based presentation generation
- Architecture documentation workflows

**Note:** This is a specific use case example. Adapt the scripts to your own presentation needs.
